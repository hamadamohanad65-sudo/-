# نظام الشات (.شات)

نظام ذكاء اصطناعي سحابي:
- **النصوص**: [Groq](https://console.groq.com/keys) - مجاني بالكامل، وسريع
  جدًا لأنه بيستخدم LPU مش GPU عادي (رد شبه فوري)، وحده المجاني للطلبات
  أعلى بكتير من Gemini.
- **الصور** (`.صورة_شات`): [Gemini](https://aistudio.google.com/apikey) -
  لأن Groq مفيهوش موديل توليد صور.

## الإعداد

محتاج مفتاحين موجودين أصلاً في `config.js` الرئيسي:
- `GROQ_API_KEY` (نفس المفتاح المستخدم في تفريغ الصوت)
- `GEMINI_API_KEY` (نفس المفتاح المستخدم في `.اسأل` و`.بحث`، للصور بس هنا)

## الملفات

| الملف | الوظيفة |
|---|---|
| `groqClient.js` | الاتصال بـ Groq (`chat/completions`) للنصوص |
| `imageGen.js` | توليد الصور عن طريق Gemini |
| `config.js` | كل الإعدادات (الموديل، الطابور، الكاش، الحدود...) |
| `memory.js` | تاريخ المحادثة والشخصية والموديل لكل مستخدم |
| `personas.js` | الشخصيات المختلفة (System Prompts) |
| `queue.js` | طابور بيحدد عدد الطلبات المتزامنة |
| `cache.js` | كاش الردود المتكررة |
| `rateLimit.js` | الحد من عدد الطلبات لكل مستخدم |
| `textSplitter.js` | تقسيم الردود الطويلة لأجزاء |
| `logger.js` | تسجيل الأحداث والإحصائيات |

## الأوامر

- `.شات <سؤال>` / `.اسأل` / `.ai` / `.gpt` - كلم الذكاء الاصطناعي
- `.اعاده_الشات` - إعادة تعيين كاملة
- `.مسح_الشات` - مسح تاريخ المحادثة بس
- `.شخصية [اسم]` - عرض/تغيير الشخصية
- `.معلومات_الشات` - إحصائياتك مع الشات
- `.حالة_الشات` - فحص هل Groq متصل وحالة الطابور/الكاش
- `.صورة_شات <وصف>` - توليد صورة بالذكاء الاصطناعي (Gemini)
